# 🚀 Générateur de Sites avec Mistral AI

Créez des sites web complets en décrivant simplement ce que vous voulez !

## ✨ Fonctionnalités

- 🤖 Génération de sites avec l'API Mistral AI
- 🎨 Multiple styles (moderne, minimaliste, créatif, corporate)
- 📱 Sites responsives automatiquement
- 👀 Prévisualisation en direct
- 💾 Téléchargement du code généré
- 🚀 Déploiement automatique sur GitHub Pages

## 🔧 Utilisation

1. Obtenez une clé API sur [Mistral AI](https://console.mistral.ai)
2. Entrez votre clé dans l'interface
3. Décrivez votre site
4. Cliquez sur "Générer"

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript
- API Mistral AI
- GitHub Pages pour l'hébergement
